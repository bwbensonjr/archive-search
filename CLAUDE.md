## Project Overview

This repository is for searching and cataloging Internet Archive resources. The primary tool is the [Internet Archive CLI (`ia`)](https://archive.org/developers/internetarchive/cli.html), which is installed and authenticated on the local machine.

## Key Commands

- `ia search <query>` — search the Internet Archive catalog
- `ia metadata <identifier>` — get metadata for a specific item
- `ia list <identifier>` — list files in an item

## Current Focus

Cataloging the Aadam Jacobs collection (https://archive.org/details/@aadam_jacobs_collection), extracting venue, date, and band name for each show.
